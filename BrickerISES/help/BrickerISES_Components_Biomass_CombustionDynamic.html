BrickerISES.Components.Biomass.CombustionDynamic.PartialCombustionDynamic

CombustionDynamicCommonInterfaces

Information

Model PartialCombustionDynamic<> is a partial model for simulating the dynamic of a Biomass combustion process. <\p>

It takes as an input a real value comprises between 0 and 1 and it outputs the combustion power in W. <\p>

Modelica definition

partial model PartialCombustionDynamic "CombustionDynamicCommonInterfaces" Modelica.Blocks.Interfaces.RealInput Modulation "0-1 of Nominal"; Modelica.Blocks.Interfaces.RealOutput Qcombustion; /* Parameters */ parameter Modelica.SIunits.Power QcombustionNominal=1e6 "Nominal Biomass Boiler Power"; parameter Modelica.SIunits.Time tStartup(min=1)=40*60 "Time required to start up the combustion from 0 to Nominal Power [sec]"; parameter Modelica.SIunits.Power QcombustionMin=0 "Minimum Biomass Boiler Power"; parameter Modelica.SIunits.Power QcombustionStart=0 "Start value of Biomass Boiler Power"; parameter Modelica.Blocks.Types.Init CombustionInitialization=Modelica.Blocks.Types.Init.NoInit "Type of initialization (1: no init, 2: steady state, 3/4: initial output)"; final parameter Real XModulation = (QcombustionNominal-QcombustionMin)/QcombustionNominal "Regulation Fraction"; end PartialCombustionDynamic;
